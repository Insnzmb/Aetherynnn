LOCAL AI RUNTIME (user-installed)
This build is designed to work with:

- Ollama (local model runtime) — https://ollama.com/

Default model target (configured in .env / Modelfile):
- Llama 3.1 (example: llama3.1:8b)

Required notice when distributing Llama materials:
“Llama 3.1 is licensed under the Llama 3.1 Community License, Copyright © Meta Platforms, Inc. All Rights Reserved.”

Optional narrator mode:
- Grok (xAI). If you publish content generated by Grok, attribute it as “Written with Grok” or “Created with Grok”.
