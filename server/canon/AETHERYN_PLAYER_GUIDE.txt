AETHERYN — PLAYER GUIDE (READ THIS FIRST)
Release Bundle V12.3.x (2026-02)

============================================================
WELCOME
============================================================
AETHERYN is a fantasy adventure you play with an AI Game Master.
You tell the AI what you do. The AI tells you what happens.

Sometimes the AI can mess up.
Good news: this game is built so the AI can check itself and fix mistakes.

I’m sorry for any inconvenience if the AI ever gets weird or wrong.

As AI gets better, this game will get better too.

============================================================
THE ONLY THINGS YOU MUST REMEMBER
============================================================
1) Start a new chat for a new game.
2) Upload the game files.
3) Type: Start game
4) Save often.

============================================================
YOUR MAIN COMMANDS
============================================================
Start game
- Starts a new game (or starts Intake).

SAVE NOW
- Makes a “save bundle” the AI will give you.
- Copy/paste that save bundle into a safe place (notes app, email draft, etc).

RESTORE
- Paste your save bundle into a NEW chat and type: RESTORE

EXPORT BOOK
- Asks the AI to turn the resolved story so far into clean “book style” text.
  (This does NOT change your save. It’s just a nice export.)

AUTOSAVE ON / AUTOSAVE OFF
- Turns autosave on or off (when the rules allow it).

============================================================
WHAT IS “INTAKE”?
============================================================
Intake is the setup questions.
The AI must ask ALL Intake questions at once.
You answer them.
When all are answered, the story begins.

IMPORTANT:
This release requires you to be **18 or older** to play.

============================================================
HOW TO PLAY (SIMPLE)
============================================================
AI: describes the scene and gives you choices.
YOU: pick a choice (or do Freeform).
AI: rolls dice (or asks you to roll) and tells you what happens.
Repeat until you’re unstoppable (or heroically doomed).

============================================================
WHEN THE AI MESSES UP (HOW TO FIX IT)
============================================================
If something feels wrong, type:
“Please check for drift and correct yourself.”

If the AI forgot something you said, just remind it plainly:
“No, I said X earlier. Please correct the state.”

If the AI still won’t behave, do this:
1) SAVE NOW
2) Start a NEW chat
3) Paste the save bundle
4) Type: RESTORE

============================================================
SAVING TIPS (DO THIS)
============================================================
- Save at the end of big moments.
- Save before risky choices.
- Save before you stop playing.
- Keep 2 copies of your save bundle (just in case).
